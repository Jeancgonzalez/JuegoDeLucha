# Juego de Lucha ⚔️🎮

## Descripción general

Juego de combate desarrollado en Java con el objetivo de aplicar conceptos de Programación Orientada a Objetos y patrones de diseño. El proyecto permite la interacción entre distintos tipos de personajes y armas, simulando enfrentamientos donde cada personaje posee características y habilidades específicas.

La aplicación está orientada a:

- Estudiantes de programación e ingeniería de software.
- Personas interesadas en aprender Programación Orientada a Objetos.
- Usuarios que deseen explorar la implementación de patrones de diseño en Java.

---

## Funcionalidades principales

- **Sistema de personajes**
  - Creación de diferentes tipos de personajes.
  - Cada personaje cuenta con atributos y habilidades propias.

- **Sistema de armas**
  - Implementación de diversas armas para el combate.
  - Asignación de armas a los personajes.

- **Sistema de combate**
  - Interacción entre personajes durante los enfrentamientos.
  - Gestión de ataques y daño.

- **Patrones de diseño**
  - Uso del patrón Factory para la creación de armas.
  - Código organizado para facilitar su mantenimiento y expansión.

---

## Tecnologías utilizadas

- **Java**
- **Programación Orientada a Objetos (POO)**
- **Patrón de Diseño Factory**

---

## Instalación y ejecución

1. Clonar el repositorio:

```bash
git clone https://github.com/Jeancgonzalez/Juego.git
cd Juego
```

2. Compilar y ejecutar el proyecto:

```bash
javac *.java
java Main
```

> Nota: El nombre de la clase principal puede variar según la estructura del proyecto.

## Diseño y arquitectura

- Uso de encapsulamiento, herencia y polimorfismo.
- Separación de responsabilidades entre personajes, armas y lógica de combate.
- Estructura preparada para incorporar nuevos personajes y armas de forma sencilla.

---

## Objetivo del proyecto

Desarrollar una simulación de combate que permita aplicar conceptos fundamentales de Ingeniería de Software, Programación Orientada a Objetos y patrones de diseño en Java.
