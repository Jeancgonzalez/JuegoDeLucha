Juego de Lucha ⚔️🎮
Descripción general

Juego de combate desarrollado en Java como proyecto académico para poner en práctica conceptos de Programación Orientada a Objetos y patrones de diseño.

El proyecto permite la creación de diferentes tipos de personajes, cada uno con características y habilidades particulares, así como el uso de distintas armas para participar en combates. La arquitectura del sistema está diseñada para facilitar la extensión y mantenimiento del código mediante el uso de clases, herencia, polimorfismo y patrones de creación.

Funcionalidades principales
Personajes
Creación de diferentes tipos de personajes.
Atributos y comportamientos específicos para cada clase.
Sistema de combate entre personajes.
Armas
Implementación de múltiples tipos de armas.
Asignación de armas a los personajes.
Gestión de armas mediante el patrón Factory.
Programación Orientada a Objetos
Encapsulamiento de atributos y métodos.
Uso de herencia para especializar personajes y armas.
Aplicación de polimorfismo para el manejo flexible de objetos.
Tecnologías utilizadas
Java
Programación Orientada a Objetos (POO)
Patrón de Diseño Factory
Objetivo del proyecto

Desarrollar una aplicación que simule un sistema de combate entre personajes, aplicando buenas prácticas de programación y conceptos fundamentales de Ingeniería de Software.
